## How to Use

1. Click **"Choose an image"**
2. Select your photo
3. Choose grid size (6 / 9 / 12)
4. Click **Cut**
5. Click each square to download individually  
   or click **Download all** to get every piece automatically

## Self-hosting / Run locally

This is a **static HTML + CSS + JavaScript** project ‚Äî no backend needed.

### Option 1: GitHub Pages (easiest & free)

1. Fork or clone this repository
2. Make sure the main file is named `index.html`
3. Go to **Settings ‚Üí Pages**
4. Set Source ‚Üí Branch: `main` / Folder: `/ (root)`
5. Wait 1‚Äì2 minutes ‚Üí your site will be live at
